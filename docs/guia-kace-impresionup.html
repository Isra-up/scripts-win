Paso 1 — Crear el script en KACE
Scripting › Scripts › Seleccionar acción › Nueva

Llena los campos del formulario con estos valores exactos:

Campo en KACEValor a ingresar
NombreInstall-IMPRESIONUP
HabilitadoSí
TipoOnline KScript
Sistema operativoWindows
NotasInstala cola IMPRESIONUP — Kyocera TASKalfa MZ2501ci KX — IP 10.1.6.63
Paso 2 — Subir dependencias
Scripting › Scripts › Install-IMPRESIONUP › Dependencias

Agrega solo 2 archivos como dependencias. KACE los copiará a $(KACE_DEPENDENCY_DIR) en cada equipo. El script extrae el ZIP en runtime.

install-impresionup-kace.ps1  ← el script principal
Kyocera_64bit.zip  ← todos los archivos del driver comprimidos
Cómo preparar el ZIP
Abre la carpeta Kyocera_64bit\, selecciona todos los archivos (no la carpeta en sí) y comprime la selección. Verifica que OEMSETUP.INF quede en la raíz del ZIP — si queda dentro de una subcarpeta el script fallará.
Paso 3 — Configurar la tarea
Install-IMPRESIONUP › Tareas

Configura las secciones Verificar y Correctamente de la tarea:

A
Sección: Verificar
Comprueba que el archivo de driver llegó correctamente al equipo antes de ejecutar el script.

B

Se ejecuta si la verificación fue exitosa. Lanza el script PowerShell en el equipo.

Esperar finalizaciónSí obligatorio
Ejecutar comoLocal System
Parámetros — copia exactamente -ExecutionPolicy Bypass -NonInteractive -File "$(KACE_DEPENDENCY_DIR)\install-impresionup-kace.ps1"
Nota
La sección Corrección puede dejarse vacía — si la verificación falla (no encuentra Kyocera_64bit.zip), el script no se ejecuta y KACE registra el fallo. Verifica que el ZIP esté subido correctamente en la pestaña Dependencias.
Paso 4 — Asignar equipos objetivo
Install-IMPRESIONUP › Etiquetas de máquina

Selecciona la etiqueta o grupo de equipos donde se instalará IMPRESIONUP. Guarda el script después de asignar.

Paso 5 — Ejecutar el despliegue
Scripting › Scripts › Ejecutar ahora

Selecciona Install-IMPRESIONUP en la lista → Seleccionar acción → Ejecutar ahora. El script se desplegará en todos los equipos de la etiqueta asignada en cuanto el agente KACE haga check-in.

Tiempo de check-in
El agente KACE hace check-in cada 4–8 horas por defecto. Para ejecución inmediata, fuerza el check-in desde el equipo con el agente KACE o ajusta el intervalo en la configuración.
Paso 6 — Verificar resultados
Scripting › Script Logs

Filtra por Install-IMPRESIONUP. La columna Exit Code indica el resultado por equipo:

0
Éxito
IMPRESIONUP instalada correctamente en el equipo
1
Error
Revisar el log en C:\ProgramData\KaceScripts\logs\ del equipo afectado

Para diagnóstico detallado, accede al equipo y abre el archivo de log:

Ruta del log en el equipo C:\ProgramData\KaceScripts\logs\impresionup_<NOMBRE-EQUIPO>_<fecha_hora>.log
